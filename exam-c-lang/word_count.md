# 문제 1: 단어 세기

표준 입력으로 텍스트가 주어진다. 텍스트의 **줄 수**, **단어 수**, **문자 수**를 구하시오.

단어는 공백(` `), 탭(`\t`), 줄바꿈(`\n`)을 포함하지 않는 연속된 문자열로 정의한다.

## 입력

여러 줄의 텍스트가 주어진다. (EOF까지 읽는다)

## 출력

줄 수, 단어 수, 문자 수를 공백으로 구분하여 한 줄에 출력한다.

## 예제 입력 1

```
hello world
```

## 예제 출력 1

```
1 2 12
```

## 예제 입력 2

```
the quick brown
fox jumps over
the lazy dog
```

## 예제 출력 2

```
3 9 45
```

## 예제 입력 3

```
  hello   world  
```

## 예제 출력 3

```
1 2 19
```

## 힌트

- 문자 수에는 공백, 탭, 줄바꿈도 포함된다.
- 연속된 공백이 있어도 단어 수는 변하지 않는다.
- 빈 입력이면 `0 0 0`을 출력한다.
