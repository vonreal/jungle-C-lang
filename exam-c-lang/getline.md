# 문제 2: 가장 긴 줄 출력하기

표준 입력으로 여러 줄의 텍스트가 주어진다. **가장 긴 줄의 길이**와 **그 내용**을 출력하시오.

줄의 길이는 줄바꿈 문자를 포함한 길이이다.

## 입력

여러 줄의 텍스트가 주어진다. (EOF까지 읽는다)

각 줄의 길이는 1000자를 넘지 않는다.

## 출력

첫째 줄에 가장 긴 줄의 길이를 출력한다.

둘째 줄에 가장 긴 줄의 내용을 출력한다.

길이가 같은 줄이 여러 개이면 가장 먼저 나온 줄을 출력한다.

## 예제 입력 1

```
hello
world
```

## 예제 출력 1

```
6
hello
```

## 예제 입력 2

```
short
this is a longer line
medium length
```

## 예제 출력 2

```
22
this is a longer line
```

## 예제 입력 3

```
abc
def
ghi
```

## 예제 출력 3

```
4
abc
```

## 힌트

- 줄의 길이에는 줄바꿈(`\n`) 문자도 포함된다. 그래서 `hello\n`의 길이는 6이다.
- 이 문제를 풀려면 두 가지 기능이 필요하다:
  1. 한 줄을 읽어오는 함수 (`getline`)
  2. 문자열을 복사하는 함수 (`copy`)
- Python에서는 이 기능들이 내장되어 있지만, C에서는 직접 구현해야 한다. 그 차이를 의식하면서 풀어보자.
