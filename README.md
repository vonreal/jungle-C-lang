# 🌿 Krafton Jungle — Week 06~08: 탐험 준비

> C언어, 컴퓨터 구조, 시스템 프로그래밍

Assembly에 가장 가까운 언어인 C를 통해 컴퓨터의 본질에 다가가는 3주간의 여정.  
메모리를 직접 다루고, 자료구조를 밑바닥부터 구현하며, CS:APP을 따라 시스템의 동작 원리를 이해한다.

---

## 📅 주차별 로드맵

| 주차 | 핵심 주제 | CS:APP | 키워드 |
|------|-----------|--------|--------|
| **Week 06** | C언어 기초 · 자료구조 구현 | 3장 — 프로그램의 기계 수준 표현 | `C Pointer`, `동적 메모리 할당`, `Linked List`, `Stack`, `Queue`, `Binary Tree`, `BST` |
| **Week 07** | 메모리 관리 | 9장 — 가상 메모리 | `Virtual Memory`, `malloc/free`, `Memory Hierarchy` |
| **Week 08** | 네트워크 프로토콜 | 11장 — 네트워크 프로그래밍 | `Socket`, `HTTP`, `Client-Server Model` |

---

## 🎯 학습 목표

- C언어로 핵심 자료구조(Linked List, Stack, Queue, Binary Tree, BST)를 직접 구현한다.
- 포인터와 동적 메모리 할당을 통해 메모리 관리의 원리를 체득한다.
- C 코드가 어셈블리로 변환되고 실행되는 과정을 이해한다.
- 가상 메모리 시스템의 구조와 동작 원리를 파악한다.
- 소켓 프로그래밍을 통해 네트워크의 기초를 경험한다.
